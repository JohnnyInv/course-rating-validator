Course Rating Validator

Validates a custom or combination tee's Course Rating and Slope Rating against the official USGA/WHS table-adjustment procedure for unrated tees, using a nearby rated tee (from the course's scorecard) as the reference point.

Gender selects which official table applies

Reference tee already rated, from the scorecard

Target tee the custom / combo setup being rated

Result

Method: WHS Rule 7f1 / Appendix G (formerly USGA Handicap Manual Section 5-2g), “Ratings Adjustments from Unrated Tees.” Under 100 yards difference from the nearest same-gender rated tee, no adjustment is made. Between 100 and 300 yards, the official adjustment table is applied (the specific row used is shown with each result). Over 300 yards, the setup is not eligible this way and the Authorized Association must be contacted before it can be submitted.

Source: Table values confirmed directly against the USGA Rules of Handicapping, effective January 2024 (the current WHS revision as of this writing), Appendix G ("The Golf Course, Course Rating and Slope Rating"), retrieved from usga.org 2026-07-15. This 18-hole table is unchanged from the 2016 USGA Handicap Manual table for the 100–300 yard range this tool uses, with one correction: the official table's last band now runs through exactly 300 yards (the previous version of this tool cut that band off a few yards early, at 296/298, causing the wrong row to be used for yardage gaps of 297–300).

This tool does not issue official ratings and is not a substitute for a formal USGA Course Rating.
